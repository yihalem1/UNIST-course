# Information Visualization

UNIST course on information visualization. Hands-on D3.js exercises for
common chart idioms (line, stacked bar, scatter, interactive multi-line),
plus a final project building a multi-view dashboard of the World Happiness
Report (2015–2019) in Python / Dash / Plotly.

## Exercises

| # | Topic | Stack | Folder |
|---|---|---|---|
| 1 | Multi-line chart of life expectancy (top-5 countries by min/max gap) | D3.js v7 | [exercise1_line_chart/](exercise1_line_chart/) |
| 2 | Horizontal stacked bar of COVID-19 vaccination rates (top-15 countries) | D3.js | [exercise2_stacked_bar/](exercise2_stacked_bar/) |
| 3 | Interactive COVID-19 dashboard: country checkboxes, date range, line/bar toggle | D3.js v6 | [exercise3_interactive_covid/](exercise3_interactive_covid/) |
| 4 | Bubble / scatter chart over population, fertility, life expectancy | D3.js v5 | [exercise4_scatter/](exercise4_scatter/) |
